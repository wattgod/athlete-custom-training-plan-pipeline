WARM-UP:
-10min warm-up @ RPE 2-3.

MAIN SET:
-10min Easy running @ RPE 2-3.
-5x [2min Controlled downhill @ RPE 5-6 + 3min Walk back up @ RPE 2-3].
-Cadence: Fast, light feet; increase cadence rather than overstriding.
-Terrain: Smooth, moderate downhill with a safe walk-up return.

COOL-DOWN:
-5min cool-down @ RPE 1-2.

PROGRESSION:
-Level 3/6: Development - Standard training load, building consistency

PURPOSE:
Build downhill skill and eccentric tolerance with deliberate, repeatable exposure.

EXECUTION:
-Release the brakes gradually: stay in control, eyes ahead, and stop if form or footing degrades.

NUTRITION:
-None needed at this duration.

HYDRATION:
-500-750ml/hr to thirst; start hydrated and use electrolytes in warm conditions.

RPE:
-2-3/10 easy running/power-hiking, 5-6/10 race effort.
