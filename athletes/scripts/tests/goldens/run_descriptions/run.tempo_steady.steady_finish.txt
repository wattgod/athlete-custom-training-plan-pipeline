WARM-UP:
-10min warm-up @ RPE 2-3.

MAIN SET:
-29min Easy aerobic running @ RPE 3-4.
-16min Steady finish @ RPE 5-6.
-Cadence: 170-180spm; keep cadence steady as the effort rises.
-Terrain: Rolling, nontechnical route with uninterrupted final running.

COOL-DOWN:
-5min cool-down @ RPE 1-2.

PROGRESSION:
-Level 3/6: Development - Standard training load, building consistency

PURPOSE:
Raise sustainable steady-running capacity and teach a controlled strong finish.

EXECUTION:
-Let effort rise only in the final block; finish controlled, never sprinting.

NUTRITION:
-Optional: carry carbs if you are practicing race fueling or starting under-fueled.

HYDRATION:
-500-750ml/hr to thirst; start hydrated and use electrolytes in warm conditions.

RPE:
-2-3/10 recovery, 3-4/10 running, 5-6/10 running, 1-2/10 recovery.
