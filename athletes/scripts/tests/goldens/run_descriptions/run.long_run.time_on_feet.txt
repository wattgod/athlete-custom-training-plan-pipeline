WARM-UP:
-8min warm-up @ RPE 2-3.

MAIN SET:
-3x [23:15 Easy run @ RPE 3-4 + 7:45 Purposeful hike @ RPE 3-4].
-Cadence: 170-180spm on runnable ground; shorten stride on climbs.
-Terrain: Trail or rolling route with sustained climbs; practice the run/hike rhythm.
-Easy run: 70-83% LTHR.
-Purposeful hike: 70-83% LTHR.

COOL-DOWN:
-7min cool-down @ RPE 1-2.

PROGRESSION:
-Level 3/6: Development - Standard training load, building consistency

PURPOSE:
Time-on-feet durability and practiced run/hike pacing for long trail days.

EXECUTION:
-Hold the same all-day rhythm from the first climb; no late-race surge.

NUTRITION:
-40-60g/hr for Z2 endurance.

HYDRATION:
-500-750ml/hr to thirst; add sodium beyond 2 hours.

RPE:
-2-3/10 recovery, 3-4/10 running/power-hiking, 1-2/10 recovery.
