WARM-UP:
-10min warm-up @ RPE 2-3.

MAIN SET:
-5min Easy running @ RPE 3-4.
-3x [8min All-day ultra effort @ RPE 5-6 + 2min Easy reset @ RPE 2-3].
-Cadence: Economical cadence; shorten stride before effort rises.
-Terrain: Race-like rolling trail or loop with controlled climbs.

COOL-DOWN:
-5min cool-down @ RPE 1-2.

PROGRESSION:
-Level 3/6: Development - Standard training load, building consistency

PURPOSE:
Rehearse the sustainable all-day effort, pacing restraint, and fueling rhythm of an ultra.

EXECUTION:
-Cap the race blocks at all-day effort and use the recoveries to rehearse fueling and reset.

NUTRITION:
-None needed at this duration.

HYDRATION:
-500-750ml/hr to thirst; start hydrated and use electrolytes in warm conditions.

RPE:
-3-4/10 easy running, 5-6/10 race effort, 2-3/10 easy running.
