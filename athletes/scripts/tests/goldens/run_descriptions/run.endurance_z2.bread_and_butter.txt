WARM-UP:
-5min warm-up @ RPE 1-2.

MAIN SET:
-50min easy run @ RPE 3-4.
-Cadence: 170-180spm, conversational and relaxed.
-Terrain: Rolling dirt or road; keep climbs below the aerobic cap.
-easy run: 70-83% LTHR.

COOL-DOWN:
-5min cool-down @ RPE 1-2.

PROGRESSION:
-Level 3/6: Development - Standard training load, building consistency

PURPOSE:
Aerobic base work that builds durable, efficient all-day running.

EXECUTION:
-Stay conversational throughout; back off immediately if effort drifts above RPE 4.

NUTRITION:
-Optional: carry carbs if you are practicing race fueling or starting under-fueled.

HYDRATION:
-Drink to thirst; carry water when conditions or access require it.

RPE:
-1-2/10 recovery, 3-4/10 running.
