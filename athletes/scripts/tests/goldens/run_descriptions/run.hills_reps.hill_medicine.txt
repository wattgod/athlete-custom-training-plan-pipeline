WARM-UP:
-12min warm-up @ RPE 2-3.

MAIN SET:
-16min Easy running before reps @ RPE 2-3.
-8x [1:15 Uphill rep — run strong @ RPE 6-7 + 2:15 Walk down @ RPE 1-2].
-16min Easy running after reps @ RPE 2-3.
-Cadence: Powerful arms and quick feet uphill; walk-down recovery is deliberate.
-Terrain: Short, steep hill with safe walking descent.

COOL-DOWN:
-3min cool-down @ RPE 1-2.

PROGRESSION:
-Level 3/6: Development - Standard training load, building consistency

PURPOSE:
Develop uphill strength and descending resilience with controlled repeatable climbs.

EXECUTION:
-Run strong but controlled uphill; take every walk-down recovery so quality stays high.

NUTRITION:
-Optional: carry carbs if you are practicing race fueling or starting under-fueled.

HYDRATION:
-500-750ml/hr to thirst; start hydrated and use electrolytes in warm conditions.

RPE:
-2-3/10 running, 6-7/10 running, 1-2/10 running.
