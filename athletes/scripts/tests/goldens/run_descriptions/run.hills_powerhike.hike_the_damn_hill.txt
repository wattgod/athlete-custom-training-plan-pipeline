WARM-UP:
-10min warm-up @ RPE 2-3.

MAIN SET:
-5x [6min Easy run between climbs @ RPE 3-4 + 3min POWER-HIKE climb @ RPE 5-6].
-Cadence: 170-180spm running; hike with short, strong steps and active arms.
-Terrain: Rolling trail with a sustained, runnable-to-hikeable climb.

COOL-DOWN:
-5min cool-down @ RPE 1-2.

PROGRESSION:
-Level 3/6: Development - Standard training load, building consistency

PURPOSE:
Build efficient climbing strength and make purposeful hiking a race-ready skill.

EXECUTION:
-Hike hard enough that running would only be marginally faster; recover fully on the easy run.

NUTRITION:
-Optional: carry carbs if you are practicing race fueling or starting under-fueled.

HYDRATION:
-500-750ml/hr to thirst; start hydrated and use electrolytes in warm conditions.

RPE:
-2-3/10 recovery, 3-4/10 recovery, 5-6/10 power-hiking, 1-2/10 recovery.
